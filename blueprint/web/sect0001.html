1 The One-Period Binomial Model for European Contingent Claims

1.1 The Market

Definition 1
#

The initial market parameters are defined by

  • \(S_0 {\gt} 0\), the initial price of the risky asset;

  • \(u {\gt} 0\), the up factor;

  • \(d {\gt} 0\), the down factor;

  • \(r {\gt} -1\) such that the risk-free asset grows by a factor of \(1 + r\) at each time step.

Definition 2
#

Uses

  • Definition 1

The sample space for the ECC binomial model is \(\Omega = \{ u, d\} \), representing a single up or down move. We apply the probability measure \(P\) such that \(P(\{ u\} ) {\gt} 0\) and \(P(\{ d\} ) {\gt} 0\).

Definition 3
#

Uses

  • Definition 2

The filtration for the ECC binomial model is the natural filtration, defined as \(\mathcal{F}^0_0 = \{ \emptyset , \Omega \} \) and \(\mathcal{F}^0_1 = 2^\Omega \).

1.2 Asset Prices

Definition 4
#

Uses

  • Definition 1

The bond is the risk-free asset, with price process defined by \(B_0 = 1\) and \(B_1 = 1 + r\).

Definition 5
#

Uses

  • Definition 1
  • Definition 2
  • Definition 3

The stock is the risky asset, with price process defined by \(S_0\) at time 0 and

\[ S_1(\omega ) = \begin{cases} u S_0 & \text{if } \omega = u, \\ d S_0 & \text{if } \omega = d. \end{cases} \]
Definition 6
#

Uses

  • Definition 3

A trading strategy is a pair of \(\mathcal{F}^0\)-adapted processes \(\varphi = (\alpha , \beta )\), where \(\alpha _t\) represents the number of shares of the stock held at time \(t\) and \(\beta _t\) represents the number of shares of the bond held at time \(t\). The value of the trading strategy at time \(t\) is given by \(V_t(\varphi ) = \alpha _t S_t + \beta _t B_t\).

1.3 Arbitrage

Definition 7
#

Uses

  • Definition 6
  • Definition 4
  • Definition 5

There exists arbitrage or equivalently an arbitrage opportunity if there exists a trading strategy \(\varphi = (\alpha , \beta )\) such that

  • \(V_0(\varphi ) = 0\);

  • \(V_1(\varphi ) \geq 0\);

  • \(E[V_1(\varphi )] {\gt} 0\).

Definition 8
#

Uses

  • Definition 7

The market is arbitrage-free if there are no arbitrage opportunities.

Definition 9
#

Uses

  • Definition 1
  • Definition 2
  • Definition 3
  • Definition 4
  • Definition 5

A risk-neutral probability is a probability measure \(q\) on \(\Omega \) such that

\[ q = \frac{(1 + r) - d}{u - d} \]
Lemma 10
#

Uses

  • Definition 1
  • Definition 8

The market is arbitrage-free if and only if \(d {\lt} 1 + r {\lt} u\).

Proof ▶

  • (\(\Rightarrow \)) Assume the market is arbitrage-free. If \((1 + r) \leq d\), then define the trading strategy \(\varphi = (\alpha , \beta )\) by \(\alpha _0 = 1\), \(\beta _0 = -S_0\). Then \(V_0(\varphi ) = 0\) and \(V_1(\varphi ) = S_1 - S_0 (1 + r) \geq 0\) with positive probability. Thus, \(E[V_1(\varphi )] {\gt} 0\), which contradicts the assumption of no arbitrage. Similarly, if \((1 + r) \geq u\), then define the trading strategy \(\varphi = (\alpha , \beta )\) by \(\alpha _0 = -1\), \(\beta _0 = S_0\). Then \(V_0(\varphi ) = 0\) and \(V_1(\varphi ) = -S_1 + S_0 (1 + r) \geq 0\) with positive probability, leading to a similar contradiction.

  • (\(\Leftarrow \)) Assume \(d {\lt} 1 + r {\lt} u\). Let \(\varphi = (\alpha , \beta )\) be any trading strategy such that \(V_0(\varphi ) = 0\). Then \(\beta _0 = -\alpha _0 S_0\) and

    \[ V_1(\varphi ) = \begin{cases} \alpha _0 S_1 + \beta _0 S_0 (1 + r) = \alpha _0 S_0(u - (1 + r)) & \text{if } \omega = u, \\ \alpha _0 S_1 + \beta _0 S_0 (1 + r) = \alpha _0 S_0(d - (1 + r)) & \text{if } \omega = d. \end{cases} \]

    Since \(d {\lt} 1 + r {\lt} u\), we note that \(V_1(\varphi ) {\lt} 0\) with positive probability, so the market is arbitrage-free.

Lemma 11
#

Uses

  • Definition 9
  • Definition 1

The risk-neutral probability \(q\) is in the unit interval \((0, 1)\) if and only if \(d {\lt} 1 + r {\lt} u\).

Proof ▶

Follows directly from the definition of \(q\) and the condition \(d {\lt} 1 + r {\lt} u\).

Definition 12
#

Uses

  • Definition 9
  • Definition 2

The risk-neutral measure \(Q\) on \(\Omega = \{ u, d\} \) is defined by \(Q(\{ u\} ) = q\) and \(Q(\{ d\} ) = 1 - q\).

Lemma 13
#

Uses

  • Definition 12
  • Lemma 11
  • Lemma 11

\(Q\) is a probability measure on \((\Omega , \mathcal{F})\).

Proof ▶

We have \(Q(\{ u\} ) + Q(\{ d\} ) = q + (1-q) = 1\) and both weights are in \((0,1)\) by 11.

Lemma 14
#

Uses

  • Definition 12
  • Definition 5
  • Definition 4
  • Definition 12
  • Definition 5
  • Definition 9

The discounted stock price satisfies

\[ E^Q\! \left[\frac{S_1}{1 + r}\right] = S_0. \]
Proof ▶

Direct computation: \(E^Q[S_1/(1 + r)] = q \cdot u S_0 / (1 + r) + (1-q) \cdot d S_0 / (1 + r) = S_0(qu + (1-q)d)/(1 + r) = S_0 \cdot (1 + r)/(1 + r) = S_0\).

1.4 European Contingent Claims

Definition 15
#

Uses

  • Definition 2

A European Contingent Claim (ECC) is a random variable \(\Phi : \Omega \to \mathbb {R}\). We write \(\Phi _u = \Phi (u)\) and \(\Phi _d = \Phi (d)\) for its two possible payoffs.

Example 1.1
#

Uses

  • Definition 15
  • Definition 5

The European call option with strike \(K {\gt} 0\) has payoffs \(\Phi _u = \max (uS_0 - K, 0)\) and \(\Phi _d = \max (dS_0 - K, 0)\).

Example 1.2
#

Uses

  • Definition 15
  • Definition 5

The European put option with strike \(K {\gt} 0\) has payoffs \(\Phi _u = \max (K - uS_0, 0)\) and \(\Phi _d = \max (K - dS_0, 0)\).

1.5 Replicating Strategies

Definition 16
#

Uses

  • Definition 6
  • Definition 15

A trading strategy \(\varphi = {\alpha , \beta }\) is a replicating strategy for an ECC \(\Phi \) if \(V_1(\varphi ) = \Phi \).

Theorem 17
#

Uses

  • Definition 16
  • Lemma 10
  • Definition 16
  • Lemma 10

Under the arbitrage-free condition, every ECC \(\Phi \) has a replicating strategy.

Proof ▶

The replication conditions form a linear system in \((\alpha , \beta )\) whose coefficient matrix has determinant \((u - d)S_0 (1 + r) \neq 0\) by 10. Solving by substitution yields the stated formulas.

1.6 Pricing

Theorem 18
#

Uses

  • Theorem 17
  • Definition 12
  • Definition 15
  • Definition 6
  • Theorem 17
  • Definition 9
  • Definition 12

The initial value of the unique replicating strategy of an ECC \(\Phi \) is

\[ V_0 = \frac{1}{1 + r}\, E^Q[\Phi ] = \frac{q\, \Phi _u + (1-q)\, \Phi _d}{1 + r}. \]
Proof ▶

Substituting the formulas from 17 into \(V_0 = \alpha S_0 + \beta \) and simplifying using \(q = ((1 + r) - d)/(u - d)\) yields \(V_0 = (q\Phi _u + (1-q)\Phi _d)/(1 + r) = E^Q[\Phi ]/(1 + r)\).

Corollary 19
#

Uses

  • Theorem 18
  • Example 1.1

The arbitrage-free price of a European call option with strike \(K\) is

\[ V_0 = \frac{q\max (uS_0-K,\, 0) + (1-q)\max (dS_0-K,\, 0)}{1 + r}. \]
Corollary 20
#

Uses

  • Theorem 18
  • Example 1.1
  • Example 1.2

The prices of a European call \(C_0\) and put \(P_0\) with the same strike \(K\) satisfy put-call parity:

\[ C_0 - P_0 = S_0 - \frac{K}{1 + r}. \]
